# Minimalist Help

Answer in under 15 lines:
- What: subtraction-first engineering skill; smallest correct change wins.
- The descent: delete requirement → reuse codebase → stdlib → platform → installed dep → one line → minimal new code.
- Guardrail: validation, auth, error handling, cleanup, tests are never cut.
- Levels: `/minimalist lite|full|ultra|off` (default full).
- Commands: `/minimalist`, `/minimalist-review`, `/minimalist-audit`, `/minimalist-gain`, `/minimalist-help`.
- Coexistence: yields prose style to output skills (caveman etc.); never emits conflicting instructions.
- Honesty: reports only measured savings.
